# Healthcare Waitlist Analysis Dashboard (Power BI)

## 📊 Project Overview

This project analyzes healthcare waitlist data including inpatient, outpatient, and day-case trends over time.

## 🔍 Key Insights

* Monthly waitlist trends
* Case type comparison
* Age profile analysis
* Wait time distribution

## 🛠 Tools Used

* Power BI
* Power Query
* DAX
* Data Modeling
